# Student Result Management System

A Java console-based application to manage student academic records and results.

## Overview
This project allows storing student details, processing subject-wise marks,
and calculating total, average, percentage, and grade using Java.

## Features
- Student information management
- Marks input and result calculation
- Grade and percentage generation
- File handling using text files

## Technologies Used
- Java
- Object-Oriented Programming
- File Handling

## How to Run
1. Compile all `.java` files
2. Run the main program
3. Follow the console menu options
